Queue链表实现的注意点

注意如下几个:
enqueue尾部插入

  • 1, 旧尾结点用一个Node保存, last指向新的引用
  • 2, last结点指向下一个null结点
  • 3, 判空，if (first == null) first = last;
  • 4, 非空, 上旧尾结点与新尾结点链接

dequeue头部删除

  • 1, 判空, 抛出异常
  • 2, 保留返回值, first = first.next;删除结点
  • 3, 判空, last = null; 若删除前只剩一个结点, first.next为空,
